# Build with AI: Workshop Prototypes 🤖🚀

This repository contains the web applications I built during the **Build with AI: Builder Day – Jabalpur**, organized by **Growth Square** and **codecrax**. 

Instead of treating AI as a simple chatbot assistant, I leveraged **Google's integrated AI stack** as an active development partner to rapidly transition from raw concepts into multiple functional applications.

---

## 🛠️ Project Portfolio

### 🌌 1. Space Shooter Game (`/space-shooter`)
* **AI Generation:** Built the core game loops, asset rendering logic, score keeping, and collision mechanics using advanced Prompt Engineering inside **Google AI Studio**.
* **UI/UX:** Utilized **Google Stitch** to quickly structure a retro, high-contrast arcade interface.

### 🗺️ 2. Wayfarer App (`/wayfarer-app`)
* **AI Generation:** Leveraged detailed Context Engineering to spin up the core data states and foundational routing logic rules.
* **UI/UX:** Used **Google Stitch** for rapid component prototyping, taking features from abstract ideas to standard front-end web layouts.

---

## ⚙️ Tech Stack & Concepts Explored

* **Core Framework:** React + TypeScript + Vite
* **AI Prototyping Ecosystem:** Google AI Studio & Google Stitch
* **Scalability Paradigms:** Explored **Google Antigravity** workflows to understand how to orchestrate multi-agent architectures for highly scalable applications.
